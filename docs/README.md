글 전체를 관통하는 핵심 메시지:

- 처음에는 사용자가 PDF/URL을 입력하는 분석 도구를 생각했지만, 프로젝트 목표에 맞게 에이전트가 공식 출처를 직접 모니터링하는 구조로 바꿨다.
- 금융위 목록 HTML은 신규 규제 후보를 찾는 데 쓰고, PDF 본문은 실제 변경 내용과 근거 조문을 분석하는 데 썼다.
- 법률/규제 문서는 고정 길이 청킹보다 조문/섹션 기반 청킹이 더 중요했다.
- OpenAI 분석은 실패해도 서비스가 죽지 않도록 규칙 기반 fallback을 유지했다.
